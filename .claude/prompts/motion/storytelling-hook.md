# Storytelling Hook: Die ersten 3 Sekunden

Der Hook entscheidet über Scroll-Away oder Watch.

---

## Die 3-Sekunden-Regel

**Fakt:** 65% der Viewer scrollen in den ersten 3 Sekunden weiter.

**Ziel:** In 90 Frames (@ 30fps) muss der Viewer gefesselt sein.

---

## Hook-Strategien

### 1. Problem-Agitation
**Zeige ein Problem, das der Viewer kennt.**

```
"Noch kein Booking diesen Monat?"
[Frustrierter DJ vor leerem Kalender]

"Events verpassen nervt."
[Leere Event-Liste auf Handy]
```

**Wann:** B2B-Produkte, Pain Point klar

### 2. Curiosity Gap
**Wecke Neugier ohne alles zu verraten.**

```
"Das machen erfolgreiche DJs anders..."
[Mysterious Reveal Animation]

"Ein Feature, das alles aendert."
[Silhouette, dann Reveal]
```

**Wann:** Neue Features, Überraschungen

### 3. Bold Statement
**Mache eine starke, provokante Aussage.**

```
"DJ-Booking war noch nie so einfach."
[Confident Logo Animation]

"Die Zukunft der Club-Szene ist da."
[Dramatic Typography]
```

**Wann:** Brand-Videos, Launches

### 4. Visual Surprise
**Überrasche visuell, ohne Worte.**

```
[Glitch-Effekt, schnelle Schnitte]
[Ungewoehnliche Perspektive]
[Unexpected Movement]
```

**Wann:** TikTok, juengere Zielgruppe

### 5. Question Hook
**Stelle eine Frage, die zum Nachdenken zwingt.**

```
"Was waere, wenn du nie wieder ein Event verpasst?"
"Wie viele Bookings hast du diesen Monat verloren?"
```

**Wann:** Educational Content, Thought Leadership

---

## Hook-Anatomie

```
FRAME 0-30:   Visueller Impact (Logo, Glitch, Überraschung)
FRAME 30-60:  Text erscheint (Problem/Frage/Statement)
FRAME 60-90:  Transition zur Hauptstory
```

---

## Text-Regeln für Hooks

| Regel | Beispiel |
|-------|----------|
| Max 6 Woerter | "Dein nächstes Event wartet." |
| Kein "Wir" | ❌ "Wir haben ein neues Feature" |
| Du-Ansprache | ✅ "Dein Kalender. Immer aktuell." |
| Aktiv, nicht passiv | ✅ "Finde Events" ❌ "Events werden gefunden" |

---

## Hook-Varianten nach Zielgruppe

### DJs
- Pain: Keine Bookings, Unsichtbarkeit
- Hook: "Warum andere DJs mehr Bookings bekommen"

### Veranstalter
- Pain: Leere Venues, DJ-Suche
- Hook: "Die besten DJs sind schon hier"

### Party-Gaenger
- Pain: Events verpassen, FOMO
- Hook: "Das Event des Jahres? Du waerst fast nicht dabei"

---

## Visuelle Hook-Elemente

| Element | Wirkung | Verwendung |
|---------|---------|------------|
| Glitch | Aufmerksamkeit, edgy | TikTok, junge Zielgruppe |
| Logo Reveal | Brand, Vertrauen | Brand-Videos |
| Schnelle Schnitte | Energie, Action | Promo-Videos |
| Zoom In | Fokus, Wichtigkeit | Feature-Demos |
| Motion Blur | Speed, Dynamik | Action-Sequenzen |

---

## Anti-Patterns

```
❌ Langsames Fade-In von Logo (langweilig)
❌ "Willkommen zu unserem Video" (irrelevant)
❌ Musik ohne Visual-Aktion (verschenkt)
❌ Zu viel Text im Hook (überfordernd)
❌ Generischer Stock-Content (austauschbar)
```

---

## Qualitäts-Check

- [ ] Hook in 3 Sekunden (90 Frames)
- [ ] Klare Emotion/Reaktion ausgeloest
- [ ] Text unter 6 Woerter
- [ ] Visueller Impact vorhanden
- [ ] Zielgruppen-spezifisch
