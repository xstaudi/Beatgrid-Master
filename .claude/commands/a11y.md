# Accessibility-Audit durchfuehren

WCAG 2.1 AA Accessibility-Audit für Komponenten oder Seiten.

## Usage
```
/a11y                  # Aktuell im Editor geöffnete Datei
/a11y [component]      # Spezifische Komponente
/a11y full             # Alle Komponenten
```

**Aktuelle Datei** = aktuell im Editor geöffnete Datei (fokussiertes Tab).

## WCAG 2.1 AA Checkliste

### Perceivable
- [ ] Alt-Text für Bilder (1.1.1)
- [ ] Kontrast >= 4.5:1 (Text) (1.4.3)
- [ ] Kontrast >= 3:1 (UI) (1.4.11)
- [ ] Info nicht nur durch Farbe (1.4.1)
- [ ] Semantische Headings (H1–H6) (1.3.1)

### Operable
- [ ] Tastatur-erreichbar (2.1.1)
- [ ] Fokus sichtbar (2.4.7)
- [ ] Focus Order logisch (2.4.3)
- [ ] Keine Keyboard-Traps (2.1.2)
- [ ] Touch-Targets >= 44x44px (2.5.5)

### Understandable
- [ ] Labels für Inputs (3.3.2)
- [ ] Hilfreiche Fehlermeldungen (3.3.1)
- [ ] `lang` Attribut (3.1.1)

### Robust
- [ ] Valides HTML (4.1.1)
- [ ] ARIA korrekt (4.1.2)

## Output Format

```markdown
## A11y Audit: [Component]

### Summary
- Critical: X (Blocker)
- High: X
- Medium: X
- Low: X

### Issues
| Issue | Element | WCAG Kriterium | Fix |
|-------|---------|----------------|-----|
| [Beschreibung] | [Element] | 1.4.3 | [Lösung] |

### Passed
- [x] ...
```

**Severity:** Entspricht axe-Logik (siehe Severity-Tabelle unten).

## Quick Fixes (Tailwind Examples)

```tsx
// Fokus sichtbar
focus:ring-2 focus:outline-none

// Label + Input
<label htmlFor="x">...</label>
<input id="x">

// Button statt div
<button> statt <div onClick>

// Screen Reader Text
<span className="sr-only">...</span>
```

## Testing

1. **Nur Tastatur** - Navigation ohne Maus
2. **Browser-Zoom 200%** - Content bleibt nutzbar
3. **Screen Reader** - NVDA/JAWS/VoiceOver testen
4. **axe DevTools** - Automatisiertes Scanning

## Severity

| Level | Beschreibung |
|-------|--------------|
| Critical | Blocker - Komponente nicht nutzbar |
| High | Ernsthaftes Problem - starke Einschränkung |
| Medium | Moderates Problem - leichte Einschränkung |
| Low | Best Practice - Verbesserung möglich |
